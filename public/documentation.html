Movies Application

This is the server-side component of a "movies" web application.
This API provides the users with a list of movies.
The users can so have a look on a list of available movies, get all detaild information about a single movie, genre as well as director.
The users can create an account and update or delete their account. they also have the possibility to make a list of their favorite movies or remove from the favorit list.
It includes 4 general url (users, movies, directors, and genres)
